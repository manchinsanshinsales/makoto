# 接客英語の動画マニュアル構成フォーマット

現場のスタッフ（初心者）がスマホなどで隙間時間に視聴し、即座に実践できる動画マニュアルの推奨フォーマットと具体例です。

---

## 1. 推奨動画フォーマット
- **動画の長さ**: 1本あたり **1分〜2分**（最大でも3分以内）
  - ※長すぎると見返したくなくなり、必要なフレーズを探すのが困難になります。
- **画面アスペクト比**: **9:16（縦型動画）** がおすすめ
  - スタッフが自分のスマートフォンで勤務前や移動中に片手で手軽に見られるため。
- **構成スタイル**: 「NG例 ➡️ GOOD例の実演ロールプレイ」＋「字幕付きキーフレーズ解説」

---

## 2. 標準的な動画の構成（テンプレート）

| パート | 時間 | 内容 | 画面イメージ |
| :--- | :--- | :--- | :--- |
| **① タイトル** | 5秒 | 今回練習するシーンを提示 | 「席へのご案内編」などの文字テロップ |
| **② NGパターン** | 10秒 | やりがちな「冷たい・不自然な」対応 | 笑顔がなく、単語だけで話す様子 |
| **③ GOODパターン** | 20秒 | 笑顔とジェスチャーを交えた手本 | 丁寧なフレーズでスムーズに案内する様子 |
| **④ キーフレーズ解説** | 20秒 | 発音や言葉選びのポイントを解説 | 英語、カタカナ、日本語訳を画面に大きく表示 |
| **⑤ アフターコール** | 5秒 | 一緒に発音してみる（シャドーイング） | 「Let's try!」の声かけと練習時間 |

---

## 3. 具体例（シーン：お席へのご案内）

### タイトル: 「お客様をお席へご案内しよう！」
- **NGシーン (悪い例)**
  * スタッフが無表情で `Follow me.`（ついてきて）とだけ言ってスタスタ歩いていってしまう。
  * *解説*: `Follow me.` は少し命令口調に聞こえるため、お客様が威圧感を感じてしまいます。
- **GOODシーン (良い例)**
  * スタッフが笑顔で手を差し出し、 `This way, please. (こちらへどうぞ)` と言って、お客様の歩幅に合わせて歩く。
- **キーフレーズ解説**
  * **This way, please.**（ディス・ウェイ・プリーズ）
    - 「こちらへどうぞ」の定番フレーズ。手を斜め前に差し出すジェスチャーを加えるとより親切です。
  * **Please follow me.**（プリーズ・フォロー・ミー）
    - `Follow me` に `Please` をつけるだけでも丁寧になります。
- **練習時間**
  * 画面に「Repeat after me: This way, please!」と表示され、無音（または小さなBGM）でスタッフが発音する間を作る。

---

## 4. 決定した制作方針：AI音声の活用
今回の動画マニュアル作成にあたっては、以下のハイブリッド方式を採用します。

- **映像**: スタッフが親しみを持てるよう、実際の現場（または担当者様）の映像・ジェスチャーを実演。
- **音声（英語セリフ部分）**: Google Cloud Text-to-Speech などの高品質なAI音声（ネイティブ発音）を生成し、映像に重ねる。

これにより、英語の発音に不安がある場合でも、プロフェッショナルで聞き取りやすいネイティブ発音の動画教材を簡単に作成できます。
